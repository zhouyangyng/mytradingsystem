本项目来自 Codex 旧线程《分析A股市场结构》（线程 ID：`019eb684-6e67-7cb0-b980-81b3249a1c95`），后续作为长期迭代的个人市场环境/多空系统使用。

## 当前项目定位

这是一个个人使用的指数环境判断系统，核心目标不是预测每天涨跌，而是判断当前市场是否适合重仓。

系统用 `多 / 转 / 空` 三个状态描述指数环境：

- `多`：环境支持进攻，可提高仓位。
- `转`：方向不明，观察期，只适合控制仓位试错。
- `空`：不适合重仓，需要降低仓位。

当前主判指数固定为 **中证全指 `sh000985`**。选择它的原因是：它覆盖全A市场，比上证指数更能代表整体赚钱效应；同时可通过腾讯行情接口稳定自动化获取历史日K。

## 状态评分规则

每日总分为 100 分，按收盘后数据计算。

趋势 40 分：

- 收盘价 > MA5：+10
- 收盘价 > MA10：+10
- 收盘价 > MA20：+10
- MA5 > MA10 > MA20：+10

强弱位置 25 分：

- 收盘价创近5日收盘新高：+10
- 近3日低点整体抬高：+5
- 收盘位于当日振幅上 60% 区域：+5
- 当日涨幅为正：+5
- 放量冲高回落，收盘位于当日振幅下 40%：-10

量能 20 分：

- 成交量 > MA5：+8
- 成交量 > MA20：+6
- 上涨日放量：+6
- 下跌日放量：-8

风险缓冲 15 分：

- 默认给 15 分风险缓冲。
- 收盘跌破 MA10：-5
- 收盘跌破 MA20：-10
- 单日跌幅 <= -1.5% 且放量：-10
- 近3日累计跌幅 <= -3%：-10

状态映射：

- `多`：总分 >= 70，且没有“跌破MA20”或“放量大跌”等硬风险。
- `转`：40 <= 总分 < 70，或分数虽高但存在硬风险。
- `空`：总分 < 40，或收盘跌破MA20且 MA5 < MA10。

## 主升阶段规则

满足以下任一条件，认为进入主升阶段：

- 最近5个交易日内 `多` 出现 >= 3 次。
- 最近3日为 `转, 多, 多`。
- 最近3日内出现至少2次 `转`，且最新一天为 `多`。

仓位建议：

- 主升阶段：60%-80%
- `多`但非主升：40%-60%
- `转`：20%-40%
- `空`：0%-20%，不适合重仓

## 关键决策记录

最初讨论过“全A/平均股价”作为主判指数。最终选择中证全指，是因为系统要长期自动化，数据源稳定性优先。

关于平均股价：

- 同花顺 `880003` 是更接近“平均股价”的软件商统计指标。
- 东方财富、腾讯等常见公开行情接口无法稳定获取 `880003` 历史K线。
- 同花顺历史接口存在反爬/鉴权，裸请求不稳定。
- 通达信/同花顺本地导出 CSV 是可行方案。
- 未来如拿到稳定平均股价历史源，只需替换数据适配器，状态规则和展示不变。
